# 예약현황판 DID - 안드로이드 키오스크 설정

DID(디스플레이)가 안드로이드일 때, 주소창 없이 전체화면으로 보이게 하는 방법입니다.

---

## 방법 1: PWA(홈 화면에 추가) – 가장 간단

1. **안드로이드**에서 **Chrome**으로 현황판 주소 접속  
   예: `http://서버IP:8000/display/`

2. Chrome **메뉴(⋮)** → **홈 화면에 추가** 또는 **앱 설치**  
   (또는 주소창 옆에 뜨는 "앱 설치" 배너 클릭)

3. **추가** 또는 **설치** 확인

4. 이제 **홈 화면에 생긴 "예약현황" 아이콘**으로 실행  
   → **주소창 없이** 앱처럼 전체화면으로 열립니다.

5. **(선택)** 이 앱만 쓰는 전용 DID라면  
   - 설정 → 앱 → Chrome(또는 해당 앱) → **기본값 열기** 등에서  
   - 홈 버튼/최근 앱으로 나갔다가 다시 들어오면 이 앱만 쓰게 할 수 있습니다.  
   완전 잠금은 아래 키오스크 앱 사용을 권장합니다.

---

## 방법 2: 키오스크 브라우저 앱 (전용 DID용)

DID를 **한 화면만** 보여 주는 키오스크로 쓰려면 전용 앱을 쓰는 것이 좋습니다.

### Fully Kiosk Browser (권장)

1. **Play 스토어**에서 **"Fully Kiosk Browser"** 설치

2. 앱 실행 후 **설정**에서:
   - **Start URL**: `http://서버IP:8000/display/` 입력  
     (같은 기기에서 서버 띄우면 `http://localhost:8000/display/` 또는 `http://127.0.0.1:8000/display/`)
   - **Fullscreen** / **Kiosk mode** 켜기
   - **설정 잠금**(비밀번호) 설정하면 DID 전용으로 고정 가능

3. **저장** 후 앱 재시작  
   → 항상 현황판만 전체화면으로 표시됩니다.

### 그 외

- **Kiosk Browser**, **KioWare** 등 “키오스크 브라우저”로 검색해서  
  **시작 URL**만 `http://서버IP:8000/display/` 로 넣어도 같은 방식으로 사용 가능합니다.

---

## 방법 3: Chrome만 쓰고 전체화면 시도

- 이미 현황판 페이지에서 **자동 전체화면**을 시도하는 코드가 들어가 있습니다.
- **PWA(홈 화면에 추가)**로 실행하면,  
  안드로이드에서는 주소창이 안 보이는 **standalone** 모드로 열려서  
  터치 없이도 전체화면처럼 보일 수 있습니다.
- 터치가 되는 DID면 **화면 한 번 터치** 시 전체화면이 되는 경우도 있습니다.

---

## 정리

| 방식              | 주소창 | 전체화면 | 비고           |
|-------------------|--------|----------|----------------|
| PWA(홈 화면 추가) | 없음   | 앱처럼   | 가장 간단      |
| Fully 등 키오스크 앱 | 없음   | 전체화면 | DID 전용 추천  |
| Chrome 그냥 열기  | 있음   | 시도함   | 터치 시 도움   |

**DID가 안드로이드면**  
→ 먼저 **Chrome에서 현황판 열고 → 홈 화면에 추가**해서 PWA로 실행하는 방식이 가장 쉽고,  
완전 키오스크로 쓰려면 **Fully Kiosk Browser** 같은 앱으로 `http://서버IP:8000/display/` 만 전체화면으로 띄우면 됩니다.
